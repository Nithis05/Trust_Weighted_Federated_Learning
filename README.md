# Trust Weighted Federated Learning

## Overview
This project implements a Trust Weighted Federated Learning framework where client updates are weighted based on trust scores to reduce the influence of unreliable or malicious participants.

## Technologies
Python
NumPy
Scikit-learn
Federated Learning

## Files
- trust_weighted_federated_learning.ipynb → Experimental notebook
- trust_weighted_federated_learning.py → Clean implementation script

## How to Run
1. Clone the repository
2. Install requirements
3. Run the notebook or Python script

## Results
The trust-based aggregation improves robustness compared to traditional federated averaging.
